Per Folder Preferences Settings

Specify rules here for dynamically changing Thunderbird preferences based on what folder is currently being viewed. Leaving the account or folder regexp empty matches all accounts and/or folders. The first matching rule for each preference is used. To specify a default for a particular preference, add a rule below all the others for that preference with the account and folder regexp empty.

There are two ways to find out a preference name:

  1. Search for it in the advanced preference (open Thunderbird Settings, click "General" on the left, scroll down to the bottom, click "Config Editor...").
  2. Quit Thunderbird, make a copy of prefs.js from your profile directory, open Thunderbird, change the preference in the Thunderbird settings, quit Thunderbird, and compare the copy of prefs.js you made before with the current version in your profile directory.
